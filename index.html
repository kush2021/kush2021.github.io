Nand 2 Tetris

A personal project focusing on building a computer architecture from scratch, only given a NAND logic gate. Over the course of 6 months, I built all of the logic gates, computer chips (ALU, CPU, RAM, etc.), an assembler, translator, and compiler for a high-level language, and an operating system.

Job Tracker

A hackathon project to track job applications and autofill future applications based on uploaded resume information. This application was also deployed using Docker containers onto a cloud VM instance to allow public access.

Gazprea

A full-scale compiler for a high-level programming language called Gazprea. The language included support for vectors, tuples, functions, and more. The compiler included over 20 error-checking and optimization passes using the new MLIR dialects in C++.
